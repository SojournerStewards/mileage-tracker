Google Drive Sync

One-time setup

You need a free Google API Client ID so the app can save your data to your own Drive. Takes about 5 minutes. Your data stays in your Google account — no one else (not even me) can see it.

  1. Go to console.cloud.google.com/projectcreate
  2. Create a new project (name it "Mileage Tracker" or whatever you want)
  3. In the left sidebar: APIs & Services → Library → search "Google Drive API" → click it → click Enable
  4. Then: APIs & Services → OAuth consent screen → choose "External" → fill in app name "Mileage Tracker", your email for support email and developer contact → Save and Continue through the rest (skip scopes, skip test users for now, back to dashboard)
  5. Then: APIs & Services → Credentials → + Create Credentials → OAuth Client ID
  6. Application type: Web application
  7. Under "Authorized JavaScript origins" click + Add URI and add the URL this page is running from (see below)
  8. Click Create. Copy the Client ID that appears (looks like 123456-abc...apps.googleusercontent.com)
  9. Paste it in the box below and tap Connect
